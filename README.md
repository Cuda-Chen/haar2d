# haar2d
A (inverse) Haar wavelet transform example promgram with OpenCV.

# How to Compile and Run
I use CMake to build the program.
```
mkdir build
cd build
cmake ..
make
cd ..
./haar2d
```

# Note
The version of OpenCV I use is 3.4.1.
